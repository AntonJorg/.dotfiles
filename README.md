## Dotfile tracking

### Setup on new computer
```bash
git clone --bare https://github.com/USERNAME/dotfiles.git $HOME/.dotfiles

alias dotfiles='/usr/bin/git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME'

dotfiles checkout
```
